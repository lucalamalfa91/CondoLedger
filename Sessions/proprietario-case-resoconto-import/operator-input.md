# Input operatore

Voglio aggiungere **nome e cognome del proprietario** nella configurazione della casa, in modo che durante il **caricamento documenti** (preventivo/consuntivo amministratore):

- Si selezionino **solo le mie righe** (match su nominativo configurato).
- Al termine mi mostri un **resoconto** di cosa ha trovato, ad esempio:
  - saldi precedenti
  - totali esercizio precedente
  - nuovo preventivo
  - nuovo consuntivo
  - distribuzione rate

Posso impostare **più di un proprietario** sulla stessa casa, oppure un **affittuario**: in questo caso le voci vanno **sommate** (affittuario + proprietario).

Una volta **confermato** il resoconto, i valori devono essere riportati nei **dovuti** corretti nell'app.
